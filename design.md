# Design: Kalkulator Leasingu

## Style
- Professional financial tool aesthetic
- Clean, minimal, no unnecessary animations
- Mobile-first PWA

## Colors
- Primary (navy): #1B2A4A
- Accent (gold): #C9A84C  
- Background: #F5F6FA
- Card: #FFFFFF
- Text: #1B2A4A
- Muted text: #6B7280
- Border: #E5E7EB
- Success: #10B981

## Typography
- Font: Poppins (headings) + Inter (body)
- Headings: bold, navy
- Body: regular, dark gray

## Layout
- Mobile: single column, cards stacked
- Desktop (lg+): 2 columns — calculator left, results + contact right
- Cards with subtle shadows, rounded corners
- Generous padding, clear spacing

## Components
- Inputs: clean borders, navy focus ring
- Buttons: navy primary, gold accent CTA
- Cards: white bg, subtle shadow
- Select/radio: custom styled, clean
